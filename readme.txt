== Description ==

This plugin creates a widget which easily displays, without becoming cluttered, all the contact details of the company/organisation.

It displays the information in tabs, so only one section is open at a time, which ensures your website does not become cluttered visually.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `business-contact-widget` folder to the `/wp-content/plugins/` directory
1. Activate the `Business Contact Widget` plugin through the 'Plugins' menu in WordPress
1. Go to widgets and add the `Business Contact Widget` to your sidebar.

== Frequently Asked Questions ==

= Is the plugin easy to setup? =

Yes simply move the Business Contact Widget into your sidebar and fill in the relevant details and your done.

= Can sections on the top be hidden? =

Yes, just do not enter any information in that section and that tab will not show up.
